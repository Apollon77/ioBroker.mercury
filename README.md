## mercury adapter for ioBroker

Receiving data from electricity meters Mercury.
Supports TCP / IP and Serial connections.

The following electricity meters are supported:
* Mercury-200
* Mercury-201
* Mercury-206
* Mercury-203
* Mercury-203.2TD
* Mercury-204
* Mercury-208
* Mercury-230
* Mercury-231
* Mercury-233
* Mercury-234
* Mercury-236
* Mercury-238
